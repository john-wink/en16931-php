# Conformance corpus

Real, schema-valid EN 16931 invoices used by `tests/Feature/ParityTest.php` to
prove parity against the official KoSIT validator.

- `en16931-cii.xml` — EN 16931 (CII) profile
- `xrechnung-cii.xml` — XRechnung 3.0 (CII) CIUS

Both files are example instances from
[horstoeko/zugferd](https://github.com/horstoeko/zugferd) (MIT License), included
here unmodified as third-party test data.
